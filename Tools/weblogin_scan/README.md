# WeblogicScan
Weblogic一键漏洞检测工具，V1.5
```

免责声明：Pia!(ｏ ‵-′)ノ”(ノ﹏<。)

提供一键poc检测，收录几乎全部weblogic历史漏洞。

V 1.4 更新日志:	[20200729]
    新增POC:CVE_2019_2890
    全新框架设计，高度封装与拟人化
    舍弃离线安装模块
    重点修复：从根本上解决脚本异常卡死问题（不同目标版本的异常通信造成）
    重点升级：从根本上解决脚本漏报误报问题（部分原因由py2->py3升级造成）
    # Not End：
    话说大家一直好奇其它同类型工具增加的CVE-2020-* 为什么一直没有在这里出现。
```
